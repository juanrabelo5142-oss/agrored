# 🍎 AgroRed - Mercado Hortofrutícola
Proyecto desarrollado como aprendiz SENA. Conecta productores locales con consumidores finales.

## 🚀 Cómo ejecutar localmente
1. Clonar el repositorio.
2. Crear entorno virtual: `python -m venv .venv`
3. Activar entorno e instalar: `pip install -r requirements.txt`
4. Ejecutar: `python app.py`

## 🛠️ Tecnologías
* **Backend:** Python / Flask
* **Frontend:** HTML5, CSS3 (Diseño Responsive)
* **Base de Datos:** SQL
